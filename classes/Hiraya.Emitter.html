    Hiraya.Emitter Class

    Extends Hiraya.Class
    Defined in: src/emitter.js:10
    Module: hiraya-core
    Parent Module: hiraya

    Hiraya.Emitter handles event-based callbacks. For example if you wish to create an event manager that dispatches data everytime a certain topic is called:

     Game.topicEmitter = Hiraya.Emitter.create({
   newTopic: function(topic) {
     this.emit('newTopic', topic);
   }
 });

 Game.topicEmitter.on('newTopic', function(topic) {
   console.log('Got a new topic:', topic);
 });

 Game.topicEmitter.newTopic('entityCreate');
