    length

    Number

    Defined in src/hiraya-core/collection.js:24

    Total elements in the collection. This gets updated whenever a new child is added or removed

    var collection = Hiraya.Collection.create();
collection.length; // -> 0
collection.add({ name: 'James' });
collection.length; // -> 1

    Default: 0
