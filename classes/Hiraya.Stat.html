    Hiraya.Stat Class

    Extends Hiraya.Class
    Defined in: src/hiraya-game/stat.js:8
    Module: hiraya-game
    Parent Module: hiraya

    Hiraya.Stat manages the values for an RPG attribute. It handles things like making sure the value doesn't exceed its maximum capacity.

    It also provides some useful API for value manipulation.
